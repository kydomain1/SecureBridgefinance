# SecureBridgefinance - Minimalist Lifestyle & Finance Blog

A clean, minimalist static website featuring articles on fashion, health, home, travel, finance, and food.

## Features

- **Minimalist Design**: Clean lines, ample whitespace, limited color palette
- **6 Categories**: Fashion & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, Food & Beverage
- **5 Detailed Articles**: Each with high-quality images and comprehensive content
- **Search Functionality**: Find articles by keyword
- **Category Filtering**: Browse by specific categories
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Smooth Animations**: Enhanced user experience with transitions
- **Social Media Integration**: Connected social icons in footer
- **Contact Form**: Get in touch functionality
- **About Page**: Learn about the platform

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Grid, Flexbox, animations
- **JavaScript**: Vanilla JS (no frameworks)
- **Font Awesome**: Icon library for social media
- **Unsplash**: High-quality images

## How to Use

1. **Open the website**: Simply open `index.html` in a modern web browser
2. **Browse articles**: Scroll through the homepage to see all articles
3. **Filter by category**: Click category buttons to filter articles
4. **Search**: Enter keywords in the search bar or visit the dedicated Search page
   - Search results are highlighted with matching text
   - Filter results by category
   - All results display on one page
5. **Read articles**: Click on any article card to view full content
6. **Contact**: Use the contact form to send messages
7. **Privacy**: Review the privacy policy for data handling information

## Design Philosophy

The website follows minimalist design principles:

- **Breathing Space**: Generous margins and padding
- **Limited Palette**: Black, white, and subtle grays
- **Clean Typography**: System fonts for fast loading
- **Functional Focus**: Every element serves a purpose
- **Visual Hierarchy**: Clear content organization

## Customization

To add more articles:

1. Open `js/articles-data.js`
2. Add new article object to the `articlesData` array
3. Follow the existing structure
4. Use Unsplash URLs for images

To modify colors:

1. Open `css/style.css`
2. Edit CSS custom properties in `:root`
3. Changes will apply site-wide
